# AI-Powered Placement ERP System

## Overview

AI-Powered Placement ERP System is a web-based enterprise platform developed for managing campus placement activities efficiently. The system provides dedicated dashboards for students, recruiters, and Training & Placement Officers (TPOs) while integrating AI-powered resume parsing and recommendation functionalities.

The platform automates placement workflows including student profile management, recruiter interactions, resume analysis, and job recommendations using NLP techniques.

---

## Features

- Student, Recruiter, and TPO dashboards
- AI-powered resume parsing
- Automated profile generation
- Job recommendation system
- Role-based authentication
- Placement drive management
- Resume upload and analysis
- REST API integration
- Real-time data management

---

## AI Integration

The system integrates Natural Language Processing (NLP) techniques for intelligent resume analysis and candidate profiling.

### Implemented AI Features

- Resume parsing using PyResParser
- NLP-based skill extraction
- Automated profile autofill
- Candidate recommendation workflows
- Text preprocessing using NLTK
